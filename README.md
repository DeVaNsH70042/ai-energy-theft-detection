AI-Powered Energy Theft Detection System
Overview

This project is a smart, hybrid AI system designed to detect electricity theft at the smart-meter level using both physics-based validation and machine learning anomaly detection.
The system compares actual household energy consumption with smart meter readings to identify discrepancies that indicate:

Meter tampering
Bypass connections
Direct illegal line tapping
Abnormal consumption behavior

Unlike basic rule-based systems, this solution combines deterministic validation with statistical anomaly detection for higher accuracy and reliability.
